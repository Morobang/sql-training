## What You'll Learn

### Subqueries & CTEs
- Scalar, row, and table subqueries
- Correlated subqueries
- EXISTS and NOT EXISTS
- Recursive CTEs
- Multiple CTEs in one query

### Database Objects
- Creating and managing views
- Stored procedures with parameters
- User-defined functions (scalar, table-valued)
- DML and DDL triggers
- When to use each object type

### Performance & Optimization
- Clustered vs non-clustered indexes
- Index design strategies
- Reading execution plans
- Query hints and optimization
- Statistics and cardinality estimation

### Advanced Patterns
- ACID transactions and isolation levels
- Deadlock prevention
- Table partitioning
- Analytic functions for BI
- SQL for data science and big data

---

## Prerequisites

Before starting the Advanced level, you should be comfortable with:
- Writing multi-table JOINs
- Using aggregate and window functions
- Filtering with WHERE and HAVING
- Understanding data types and NULL handling
- Basic database design concepts

## Learning Path

1. **Start with Subqueries & CTEs** - Build complex query logic
2. **Learn Database Objects** - Views, procedures, functions, triggers
3. **Master Performance** - Indexing, execution plans, optimization
4. **Advanced Topics** - Transactions, partitioning, analytics
