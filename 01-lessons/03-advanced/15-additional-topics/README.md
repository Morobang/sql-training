# Additional Intermediate Topics

This folder contains **deep-dive lessons** from the original SQL training materials that provide granular, detailed coverage of intermediate-level concepts.

## Structure

These lessons complement the main intermediate curriculum (lessons 01-14) with:
- **More examples** and edge cases
- **Detailed breakdowns** of complex topics
- **Additional practice** exercises
- **In-depth explanations** of theory

## Contents

| Folder | Topic | Description |
|--------|-------|-------------|
| `01-filtering-deep-dive` | Filtering Data | Condition evaluation, operators, precedence, complex filtering |
| `02-joins-deep-dive` | Querying Multiple Tables | Join mechanics, Cartesian products, ANSI syntax, multi-table joins |
| `03-set-operators-deep-dive` | Working with Sets | Set theory, UNION, INTERSECT, EXCEPT in detail |
| `04-data-manipulation-deep-dive` | Data Generation & Manipulation | String functions, number precision, date manipulation |
| `05-grouping-aggregates-deep-dive` | Grouping & Aggregates | GROUP BY mechanics, aggregate functions, HAVING clause |
| `06-subqueries-deep-dive` | Subqueries | Correlated vs non-correlated, multiple-row, multicolumn subqueries |
| `07-joins-revisited` | Advanced Joins | Outer joins deep dive, self-joins, cross joins |
| `08-conditional-logic-deep-dive` | Conditional Logic | CASE expressions in depth, searched vs simple CASE |

## How to Use

### Option 1: Parallel Learning
- Study main lesson (e.g., `02-sql-joins-basics`)
- Then explore the deep-dive (`02-joins-deep-dive`) for more examples

### Option 2: Sequential Learning
- Complete all main lessons 01-14 first
- Return here for advanced coverage of topics you want to master

### Option 3: Reference
- Use as a resource when solving complex problems
- Review specific sub-topics as needed
