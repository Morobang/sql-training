# Additional Beginner Topics

This folder contains **deep-dive lessons** from the original SQL training materials that provide extra detail and practice for beginner-level concepts.

## Structure

These lessons complement the main beginner curriculum and can be studied:
- **During** the main lessons for deeper understanding
- **After** completing the core beginner lessons for reinforcement
- **As reference** when you need detailed explanations

## Contents

| Folder | Topic | Description |
|--------|-------|-------------|
| `00-background` | SQL Foundations | Database history, relational model, SQL terminology |
| `01-intro-databases` | Introduction to Databases | What databases are, why we need them |
| `02-relational-model` | Relational Model | Tables, relationships, normalization basics |
| `03-sql-terminology` | SQL Terminology | Key terms and concepts in SQL |
| `04-creating-database-deep-dive` | Database Creation Details | Data types, table design, constraints in depth |
| `05-query-primer-deep-dive` | Query Fundamentals | Query mechanics, clauses, detailed SELECT coverage |

## How to Use

1. **Core Path**: Complete lessons 01-05 in the main beginner folder first
2. **Deep Dive**: Then explore these additional topics for more detail
3. **Reference**: Return here when you need clarification on fundamentals

---

**Tip**: These lessons have more granular breakdowns than the main curriculum. Perfect for learners who want comprehensive coverage of beginner topics!
